ACI Open Grid Instrument
Live diagnostic instrument tracking Finnish electricity system endurance pressure against the Black Period logic. Connects to Fingrid Open Data API (datasets 124, 74, 194). Computes Demand Pressure, Domestic Coverage, Stress Persistence, and EPP posture in real time. Requires API key configuration.

Note on tool status
Live data tools require a Fingrid Open Data API key, available free at data.fingrid.fi. Dataset IDs should be verified before deployment as Fingrid has changed identifiers historically. Design prototypes are not functional and are provided for reference only. Tools in this section are experimental and subject to change independently of the publication versioning cycle.
