Document Identity Header

A fixed, reusable header component that makes every ACI document institutionally identifiable at first glance — before the reader has processed a single word of content.

1 What It Is

The Document Identity Header (DIH-1) is a small, standardized block placed at the top of every ACI publication — above the title, below the navigation bar. It answers four questions instantly and without ambiguity:

Who published this?  ·  What type of document is this?  ·  What is its number and version?  ·  What is its current status?

These four facts are the minimum required for a document to function as a citable, institutional artifact. Without them, a document may contain excellent analysis and still be uncitable — because the reader has no stable reference point.

3 Implementation — Copy-Paste Block

Paste this block into any ACI document directly after the <nav> element, before the title. Change only the values in CAPS.

<!-- ACI Document Identity Header (DIH-1) -->
<div class="dih">
  <div class="dih-left">
    <span class="dih-institute">Aether Continuity Institute</span>
    <span class="dih-series">
      <span class="series-accent">SERIES TYPE</span>
      &nbsp;·&nbsp; No. NUMBER
    </span>
  </div>
  <div class="dih-right">
    <div class="dih-meta">
      Year &nbsp;<strong>YEAR</strong><br>
      Version &nbsp;<strong>VERSION</strong><br>
      Series &nbsp;<strong>WP / TN / DA</strong>
    </div>
    <div class="dih-status">
      <span class="dih-status-dot"></span>
      <span class="dih-status-text">STATUS</span>
    </div>
  </div>
</div>
<!-- /DIH-1 -->

DIH-1 requires 14 lines of HTML and ~40 lines of CSS already present in the ACI stylesheet. No new dependencies. No JavaScript.

4 Retrofit

WP-001 was published before DIH-1 was formalized. It should be retrofitted — a v1.1 update adding the header, with a single changelog line: "DIH-1 header added per ACI institutional standard." Content unchanged.

This is the versioning principle in action. The document gains traceability without pretending the standard existed before it did.
